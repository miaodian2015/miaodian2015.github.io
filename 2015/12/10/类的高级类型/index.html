JAVA 类的高级类型复习

类的高级类型
1.分别使用import static 导入 java。lang.Math类中的静态成员方法 max()和java。
lang.System类中的out成员变量，这是就可以在程序中直接引用这些静态成员，如在主
方法中的out。println()表达式以及直接额使用max()方法。
package com.lzw;
import static java.lang.Math.max;
import static java.lang.System.out;

public class ImportTest {

public static void main(String[] args) {
    // TODO Auto-generated method stub

//在主方法中可以直接 使用 這些靜太成員
out.println(“1和4的較大的值是：”+max(1,4));
}

}
