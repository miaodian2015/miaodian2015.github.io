JAVA 类和对象复习

1。static 静态方法中不可以使用this关键字
在静态方法中不可以直接调用非静态方法
在java中规定不能将方法体内的局内变量声明为 static

2.主方法是静态的，所以如要直接在主方法中调用其他方法，则该方法必须也是静态

3.对象的建立

public class CreateObject {

public static void main(String[] args) {    //主方法
    // TODO Auto-generated method stub        
        new CreateObject();          //构建对象
}
public CreateObject(){                //构造方法
    System.out.println("创建对象");
}

}

4.final 类

final 类名{ }
如果将某个类设置为final形式，则类中的所有方法都被隐式设置为final形式，但final类中的成员可以被定义成final或非final变量

5.成员内部类

public class OuterClass{ //外部类
private class InnerClass{

    //...
}

}

在外部类实例化内部类对象的引用
内部类构造方法
外部类方法，返回值为内部类引用
外部类不可以直接访问内部类成员变量
内部类的对象实例化操作 必须在外部类 或外部类 的非静态方法中实现
public class OuterClass {

innerClass in=new innerClass();            //在外部类实例化内部类对象的引用

class innerClass{                        //内部类构造方法
    int y;
    }
public innerClass doit(){                //外部类方法，返回值为内部类引用
    //int y;                         //外部类不可以直接访问内部类成员变量
    in.y=4;
    return new innerClass();
}

public static void main(String[] args) {
    // TODO Auto-generated method stub
    OuterClass out=new OuterClass();
    //内部类的对象实例化操作 必须在外部类 或外部类 的非静态方法中实现
    OuterClass.innerClass in=out.doit();
    OuterClass.innerClass in2=out.new innerClass();

}

}

package com.lzw;

interface OutInterface{ //定义一个接口
public void f(); //定义方法
}

public class InterfaceInner {

public static void main(String[] args) {
    // TODO Auto-generated method stub
        OuterClass2 out=new OuterClass2();//实例化
    //调用doit()方法，返回一个OutInterface接口
        OutInterface outinter=out.doit();
        outinter.f();
}

}
class OuterClass2{

//定义一个内部类实现OutInterface
private class InnerClass implements OutInterface{
    InnerClass(String s){        //内部类构造方法
        System.out.println(s);
    }
    public void f(){
        System.out.println("访问内部类中的f（）方法");
    }
}
public OutInterface doit(){
    return new InnerClass("访问内部类构造方法");

}

}
