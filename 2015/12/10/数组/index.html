5.数组查询

Arrays.binarySearch(Object[] a.Object key)
a：要搜索的数组 k:要搜索的值

必须在进行前 对数组进行排序（通过sort（）方法）。否，出错。

Arrays.binarySearch(Object[].a, int fromIndex ,int toIndex,Object key)

a:要进行检索的数组
fromIndex:指定范围的开始索引（包含）；
toIndex:指定范围的结束处索引（不包含）；
key:要搜索的元素

import java.util.Arrays; //导入java。util.Arrays
public class Rakel { //创建类

public static void main(String[] args) {//主方法
    // TODO Auto-generated method stub
    String str[]=new String[]{"fd","sa","hg","jj"};//创建数组
    Arrays.sort(str);                            //将数组进行排序
    int index=Arrays.binarySearch(str,0,2,"fd");
    System.out.println("fd的索引位置是："+index);
}

}

6.冒泡排序

public class BubbleSort {

public static void main(String[] args) {
    // TODO Auto-generated method stub
    int[] array={65,4,23,56,24,67};        //创建一个数组，
    BubbleSort sorter=new BubbleSort();  //创建冒泡排序的对象
    sorter.sort(array);                    //调用排序方法将
    }
public void sort(int[] array){
    for(int i=1;i<array.length;i++){
        for(int j=0;j<array.length-i;j++){
            if(array[j]>array[j+1]){
                int temp=array[j];
                array[j]=array[j+1];
                array[j+1]=temp;
            }
        }

    }
    showArray(array);                //输出冒泡排序后的数组元素


}

public void showArray(int[] array){
    for(int i:array){                    //遍历数组
        System.out.print(">"+i);


    }
    System.out.println();
}

}
